# Docs-patternlab

Pattern Lab for MindongLab Documentations.

## Prerequisites
 - npm
 - Gulp
   ```
    > npm install -g gulp-cli
   ```

## Usage

### Initialise
  ```
   > npm install
  ```

### Live Preview
  ```
   > gulp patternlab:serve
  ```

### Build
  ```
   > gulp patternlab:build
  ```
